# 나의 그림 일기장 (My Sketchbook Diary)

> **"오늘 하루는 어떤 색이었나요? 삐뚤삐뚤해도 괜찮아, 소중한 순간을 그려보아요!"**
>
> Spring Boot 3와 JPA를 활용해 만든, 따스한 **어린 시절 스케치북 감성**의 다이어리 프로젝트입니다.

---

## 프로젝트 컨셉 (Design Concept)

디지털의 차가움 대신, 책상 위에 펼쳐진 **노란 연필과 모눈종이**의 따뜻함을 디자인에 담았습니다.

* **스케치북 레이아웃:** 왼쪽의 스프링 타공 디자인과 부드러운 크림색 배경으로 실제 종이 질감을 구현했습니다.
* **SVG 문구 소품:** 별도의 이미지 파일 없이 SVG만으로 정교하게 만든 **노란 연필**과 **투명 볼펜**(잉크가 비치는 고동색 타입)이 각 페이지를 장식합니다. 작성 페이지에는 모노 스타일의 **지우개**도 함께 배치했습니다.
* **깔끔한 본문 서체:** `Noto Sans KR`(본문)과 `Noto Serif KR`(제목)을 조합하여 가독성과 정돈된 느낌을 동시에 살렸습니다.
* **다이어리 데코:** 마스킹 테이프로 붙인 듯한 카드 디자인과 삐딱하게 배치된 요소들로 생동감을 더했습니다.
* **페이지별 차별화:** 읽기 페이지는 별 패턴 배경의 미니멀 톤으로, 작성/수정 페이지는 모눈종이 배경의 스케치북 톤으로 분위기를 구분했습니다.

## 사용 기술 (Tech Stack)

| 분류 | 기술 스택 |
| :--- | :--- |

| **ORM** | Spring Data JPA |
| **Database** | MariaDB / MySQL |

| **Fonts** | Google Fonts (Noto Sans KR, Noto Serif KR) |

## 주요 기능 (Key Features)

### 1. 스케치북 목록 (Diary List)
- 모눈종이 배경 위에 마스킹 테이프로 붙여둔 듯한 일기 카드들을 한눈에 확인합니다.
- 카드마다 스프링 구멍 디테일과 마우스 오버 시 살짝 회전하는 애니메이션을 적용했습니다.
- 우측 하단의 노란 원형 버튼을 눌러 새 일기를 작성할 수 있습니다.

### 2. 일기 읽기 (Read Detail)
- 별 패턴 배경 위에 깔끔한 카드 형태로 일기 내용을 표시합니다.
- 상단의 핑크 마스킹 테이프 장식과 넉넉한 여백으로 미니멀한 읽기 환경을 제공합니다.
- 수정하기와 목록으로 돌아가기 버튼을 하단에 배치했습니다.

### 3. 새 조각 남기기 (Register)
- 줄 노트 양식의 입력창을 통해 오늘 하루를 기록합니다.
- 우측에 배치된 투명 볼펜과 모노 지우개 소품이 실제 책상 위 같은 분위기를 연출합니다.
- 딱딱한 표 형식을 벗어나, 실제 공책 줄 위에 글을 쓰는 듯한 UI를 제공합니다.

### 4. 다듬고 지우기 (Update & Delete)
- 수정 시에도 스케치북 디자인을 유지하여 일관된 사용자 경험을 제공합니다.
- 수정 버튼은 코랄 핑크, 취소 버튼은 부드러운 회색으로 배치하여 직관적인 조작이 가능합니다.
