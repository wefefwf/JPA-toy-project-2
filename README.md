# 참 잘했어요! 나의 그림 일기장 (My Sketchbook Diary)

> **"오늘 하루는 어떤 색이었나요? 삐뚤삐뚤해도 괜찮아, 소중한 순간을 그려보아요!"**
>
> Spring Boot 3와 JPA를 활용해 만든, 따스한 **어린 시절 스케치북 감성**의 다이어리 프로젝트입니다.

---

## 프로젝트 컨셉 (Design Concept)
디지털의 차가움 대신, 책상 위에 펼쳐진 **노란 연필과 모눈종이**의 따뜻함을 디자인에 담았습니다.

* **스케치북 레이아웃:** 왼쪽의 스프링 타공 디자인과 부드러운 크림색 배경으로 실제 종이 질감을 구현했습니다.
* **아기자기한 CSS 소품:** 별도의 이미지 없이 CSS만으로 정교하게 깎아 만든 **'통통한 노란 지우개 연필'**이 상세페이지와 작성 페이지를 장식합니다.
* **몽글몽글한 손글씨:** `감자꽃(Gamja Flower)`과 `개구(Gaegu)` 폰트를 사용하여 어린 시절의 순수한 감성을 살렸습니다.
* **다이어리 데코:** 마스킹 테이프로 붙인 듯한 카드 디자인과 삐딱하게 배치된 요소들로 생동감을 더했습니다.

---

## 사용 기술 (Tech Stack)

| 분류 | 기술 스택 |
| :--- | :--- |
| **Backend** | Java 17, Spring Boot 3.x |
| **ORM** | Spring Data JPA |
| **Database** | MariaDB / MySQL |
| **Frontend** | Thymeleaf, Bootstrap 5, css3|
| **Fonts** | Google Fonts (Gaegu, Gamja Flower) |

---

## 주요 기능 (Key Features)

### 1. 스케치북 목록 (Diary List) 
- 모눈종이 배경 위에 마스킹 테이프로 붙여둔 듯한 일기 카드들을 한눈에 확인합니다.
- 카드마다 스프링 구멍 디테일과 마우스 오버 시 살짝 회전하는 애니메이션을 적용했습니다.

### 2. 일기 읽기 (Read Detail) 
- 넓게 펼쳐진 스케치북 내지 레이아웃을 통해 작성한 일기를 상세히 확인합니다.
- **특수 포인트:** 우측 상단에 배치된 **입체적인 노란 연필** 소품이 실제 책상 위 같은 느낌을 줍니다.

### 3. 새 조각 남기기 (Register) 
- 줄 노트 양식의 입력창을 통해 오늘 하루를 기록합니다.
- 딱딱한 표 형식을 벗어나, 실제 공책 줄 위에 글을 쓰는 듯한 UI를 제공합니다.

### 4. 다듬고 지우기 (Update & Delete) 
- 수정 시에도 스케치북 디자인을 유지하여 일관된 사용자 경험을 제공합니다.
- 수정 버튼은 코랄 핑크, 취소 버튼은 부드러운 회색으로 배치하여 직관적인 조작이 가능합니다.
