#  참 잘했어요! 나의 그림 일기장 (Sketchbook Diary)

> **"삐뚤삐뚤해도 괜찮아, 소중한 오늘을 그려보아요!"** > Spring Boot와 JPA를 활용해 만든, 어린 시절 스케치북 감성이 물씬 풍기는 그림 일기장 프로젝트입니다.

---

##  프로젝트 컨셉
딱딱하고 차가운 메모장 대신, 책상 위에 펼쳐진 **스케치북**을 꺼내 보는 듯한 따뜻한 경험을 제공합니다.

* **아날로그 디자인:** 상단 스프링 장식과 모눈종이 배경으로 실제 일기장 느낌을 재현했습니다.
* **손글씨 감성:** '나눔손글씨 펜' 폰트를 사용하여 정겨운 분위기를 더했습니다.
* **추억의 레이아웃:** 삐딱하게 놓인 일기 카드와 그림 칸(그림 일기 양식)을 통해 동심을 자극합니다.

---

##  사용 기술 (Tech Stack)
| 분류 | 기술 스택 |
| :--- | :--- |
| **Backend** | Java 17, Spring Boot 3.x |
| **ORM** | Spring Data JPA |
| **Database** | MariaDB (MySQL) |
| **Frontend** | Thymeleaf, Bootstrap 5, CSS3 |
| **Font** | Google Fonts (Nanum Pen Script) |

---

##  주요 기능
1. **일기장 펼치기 (List)**
   - 스케치북 위에 나열된 일기 목록을 확인합니다.
   - 짝수/홀수마다 미세하게 다른 각도로 배치되어 실제 종이 같은 생동감을 줍니다.
2. **소중한 기록 읽기 (Read)**
   - 작성한 일기를 스케치북 속지 레이아웃으로 상세히 읽어봅니다.
3. **새 조각 남기기 (Register)**
   - 제목과 내용을 적어 새로운 일기 조각을 등록합니다.
4. **다듬고 지우기 (Update & Delete)**
   - 지난 기록을 수정하거나, 필요 없는 기록은 지울 수 있습니다.
